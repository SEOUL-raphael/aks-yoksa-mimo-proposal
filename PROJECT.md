# 프로젝트 설명

이 프로젝트는 AKS 역사지식 고도서 본문을 원문 중심으로 보존하고, 한국어 번역과 독음을 함께 제공해 고문헌 접근성을 높이기 위한 공개 작업입니다.

한문 원문은 검증과 인용의 기준으로 남기고, 번역 결과는 모델과 프롬프트 개선에 따라 다시 생성 가능한 파생 정보로 다룹니다. 공개 Markdown은 연구자와 일반 이용자가 GitHub에서 바로 읽을 수 있도록 구성했습니다.

## 데이터 원칙

- 원문 한문은 덮어쓰지 않습니다.
- 한국어 번역, 독음, 의미 요약은 별도 레이어로 둡니다.
- 출처 URL과 기사 ID를 함께 제공해 원문 확인이 가능하게 합니다.
- 실패 번역과 내부 로그는 공개하지 않고 로컬 작업 큐에서 재시도합니다.
- GitHub 웹 탐색성을 위해 개별 기사 파일 5만여 개 대신 책별 묶음 Markdown으로 제공합니다.
