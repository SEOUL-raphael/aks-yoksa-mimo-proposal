# AKS 역사지식 고도서 번역 스냅샷

한국학중앙연구원 장서각 역사지식의 고도서 본문텍스트를 수집하고, 한문 원문을 기준으로 한국어 번역, 독음, 의미 요약을 덧붙인 공개 스냅샷입니다.

핵심 원칙은 한문 원문을 기준 자료로 보존하고, 한국어 번역과 독음은 검색, 열람, 지식 그래프 구축을 위한 파생 레이어로 두는 것입니다.

## 공개 범위

- 공개 책 수: 343건
- 공개 기사 수: 52935건
- 공개 기준: MiniMax 성공 번역 중 번역문이 비어 있지 않은 항목
- 기준 레이어: 한문 원문
- 파생 레이어: 한국어 번역, 독음, 의미 요약, 엔티티 후보, 불확실 어휘

## 구조

- `kr/books/<book_id>/README.md`: 책별 서지, 해제 링크, 본문 묶음 목록
- `kr/books/<book_id>/part-001.md`: 여러 기사 번역을 묶은 Markdown
- `_index/books.md`: 사람이 읽는 책 색인
- `_index/parts.md`: 사람이 읽는 본문 묶음 색인
- `_index/books.jsonl`, `_index/parts.jsonl`: 작은 기계 판독용 색인
- `manifest.json`: 공개 스냅샷 메타데이터

원본 HTML, SQLite DB, 로그, 환경변수와 비밀키, 실패 번역 전문, 대용량 전역 기사 JSONL은 공개 저장소에 포함하지 않습니다.
